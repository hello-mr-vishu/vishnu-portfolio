# Vishnu Vardhan Portfolio

A modern, responsive, dark-themed personal portfolio built with React and Material UI, designed to showcase Professional Experience, Full-Stack Engineering, AI System Architectures, and Technical Skills.

**[Local Development]**
```bash
npm install
npm start
```

## Features
- **Dynamic Projects & Blogs**: Complete filtering system based on technical tags.
- **AI Systems Architecture**: In-depth deep dives into complex orchestration systems (RAG, Multi-Agent frameworks).
- **Categorized Skills Grid**: 9 specific domain categories mapping out technologies and frameworks.
- **Mobile First Design**: Fully responsive Material UI layout with Mobile Drawer Navigation.

## Built With
- React.js
- Material UI v5
- React Router DOM
- React Icons

## Deployment
Recommended Deployment on **Vercel**:
1. Connect this repository to your Vercel Dashboard
2. Vercel automatically detects `Create React App`
3. Click "Deploy" (Zero configuration needed, handles React-Router internally out of the box).
